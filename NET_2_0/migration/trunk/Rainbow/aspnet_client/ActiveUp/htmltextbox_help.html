Basic editing features

 

Bold

This tool will let you set the current text selection to Bold characters.

 

Italic

This tool will let you set the current text selection to Italic characters.

 

 

Underline

This tool will let you set the current text selection to Underline characters.

 

 

Cut

Cut the current selection and store it in the clipboard.

 

 

Copy

Copy the current selection and store it in the clipboard.

 

 

Paste

Paste the content of the clipboard on the current selection or cursor position.

 

 

Align Center

Align the current paragraph to center.

 

 

Align Left

Align the current paragraph to left.

 

 

Align Right

Align the current paragraph to right.

 

 

Align Justify

Justify the current paragraph.

 

 

Font Face

Set the font face. If a selection is active, the font will be applied to it.

 

 

Font Size

Set the font size. If a selection is active, the font will be applied to it.

 

 

Font Color

Set the font color. If a selection is active, the font will be applied to it.

 

 

Highlight

Highlight the current text selection.

 

 

Insert Rule

Insert an horizontal rule.

 

 

Indent

Indent the current paragraph.

 

 

Outdent

Outdent the current paragph.

 

 

Insert / Modify / Remove Link

Insert, modify or remove the link of the selected text range or image.

 

 

Ordered List

Start an ordered list. If in an unordered list, it will be converted into an ordered list.

 

 

Unordered List

Start an unordered list. If in an ordered list, it will be converted into an unordered list.

 

 

Paragraph

Set or change the current paragraph style.

 

 

Print

Print the content of the editor.

 

 

Subscript

Start or set the current selected text to subscript.

 

 

Superscript

Start or set the current selected text to superscript.

 

 

Strike Through

Start or set the current selected text to strike through.

 

 

Special Chars

Insert a special character from a predefined list.

 

Check Spelling

Check the spelling of the text.

Advanced editing features

 

Insert Image + Image Library & Uploading

Insert images, defined by the content manager or from a multiple directory image library with a previewing tool. Uploading new image is also supported.

 

 

Code Snippets

Inserts predefined HTML code snippets on the current selection or cursor position.

 

 

Code Cleaner

When a Word® document is pasted into the editor, this tool will clean the generated HTML code that have extra formatting information that is not useful and that take a lot of characters spaces. This will preserve bandwidth and performances.

 

 

Insert Table + Editing

Insert a table. Right clicking on any of the table in the editor will give you access to additional functions like adding/removing rows and columns, toggle table previewing mode and more.

 

 

Your tool

You can create your own tool that interact both with the editor content itself but also with others controls in the page, resources on server-side or anything you want.
