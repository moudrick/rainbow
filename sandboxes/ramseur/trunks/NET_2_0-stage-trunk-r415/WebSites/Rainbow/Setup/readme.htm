Upgrade note

If you get an error like this:

Server: Msg 446, Level 16, State 9, Procedure rb_GetModulesAllPortals, Line 9 Cannot resolve collation conflict for UNION operation. Server: Msg 446, Level 16, State 9, Procedure rb_GetModulesSinglePortal, Line 11 Cannot resolve collation conflict for UNION operation.

Or similar updating database or runnning it after update you should fix the collation on the named tables.

  • Open enterprise manager
  • Edit the named table (Modules for example)
  • Look for nvarchar fields
  • Set collation to Default database for all fields
